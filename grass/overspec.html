Navigation

Organisation
global problems
Structure

Convention

Duplication

Notation

Parsing

Chant

Combo

Deprecated
the use of a feature that is no longer welcome
Exotic

Overspec
the same constraint specified through several means
Priorities

Singleton

Underuse
